## 🏆 OpenAI, 연간 매출 250억 달러 돌파

ChatGPT 출시 약 2년 만에 OpenAI의 연간 환산 매출(ARR)이 **250억 달러**를 넘어섰다. AI 기업 중 역대 최단기 대규모 매출 달성이다. 구독·API·엔터프라이즈 계약이 고르게 성장하며 이뤄낸 결과로, 연내 IPO 밸류에이션에도 직접 영향을 줄 전망이다.

같은 날 OpenAI가 **Microsoft의 GitHub에 대항하는 자체 코드 호스팅 플랫폼 개발에 착수**했다는 단독 보도도 나왔다. AI 코딩 에이전트와 코드 저장소를 직접 통합하겠다는 전략인데, Microsoft와의 파트너십이 실제 플랫폼 독립 움직임으로 가시화된 첫 신호다.

## 🎤 Claude Code에 목소리가 생겼다 — Voice Mode 정식 출시

Anthropic이 AI 코딩 도구 **Claude Code에 Voice Mode를 출시**했다. 이제 개발자는 타이핑 없이 음성으로 코드 작성, 디버깅, 코드베이스 탐색을 지시할 수 있다. Cursor·GitHub Copilot 등 경쟁 도구보다 먼저 음성을 코딩 워크플로에 통합한 것으로, **핸즈프리 개발 환경**의 문이 열렸다는 평가다.

개발자가 걸어다니며, 혹은 칠판 앞에서 말로 코드를 짜는 장면이 이제 현실이 됐다.

## 💡 AI가 개발자의 언어 선택 자체를 바꾼다 — "편의 루프" 현상

GitHub Octoverse 2025 데이터 분석에서 흥미로운 패턴이 드러났다. **TypeScript가 전년 대비 66% 증가하며 GitHub 최다 사용 언어 1위**로 올라섰다.

GitHub Developer Advocate Andrea Griffiths가 이를 **"편의 루프(Convenience Loop)"**로 설명한다:
- AI가 특정 언어를 더 쉽게 만든다
- 개발자가 그 언어로 몰린다
- 학습 데이터가 쌓인다
- AI가 그 언어를 더 잘 다룬다
- 다시 처음으로

정적 타입 언어가 유리한 이유도 명확하다. LLM 컴파일 에러의 94%가 타입 체크 실패라는 연구 결과가 있으며, TypeScript의 강한 타입 시스템이 AI 생성 코드 오류를 사전에 차단한다. **AI 친화적 생태계가 곧 주류 생태계**가 되는 속도가 빨라지고 있다.

## 🇨🇳 중국 5개년 계획: 경제 전 분야에 AI 전면 도입

3월 5일 중국 전국인민대표대회 개막에 맞춰 발표된 141페이지 분량의 신규 5개년 계획에 **'AI'가 50회 이상 언급**됐다. "AI+ 행동계획"을 핵심 축으로 제조·물류·교육·의료 전 분야에 AI 에이전트 및 로봇 도입을 추진한다.

특히 **DeepSeek의 성과를 배경으로 오픈소스 AI를 미국 대비 경쟁 전략으로 공식화**한 점이 주목된다. 양자컴퓨팅·6G·휴머노이드 로봇·뇌-기계 인터페이스까지 아우르는 사실상의 기술 독립 선언이다.

## 🤖 휴머노이드 로봇, 자동차 공장에 실전 투입 시작

BMW 라이프치히 공장과 중국 주요 자동차 제조사에 휴머노이드 로봇이 **실제 조립 공정에 투입**되기 시작했다. 연구 시연을 넘어 실제 생산 라인에서 운영 가능성이 확인된 것으로, AI+로봇공학의 산업 현장 적용 속도가 눈에 띄게 빨라지고 있다.

## 🔧 에이전틱 AI, B2B 복잡 영역까지 침투 — Colt+Microsoft 사례

Colt가 Microsoft와 협력해 기업 대상 광역 네트워크 인프라 견적 산정을 **"수 일 → 약 10분"으로 단축**하는 에이전틱 AI 엔진을 공개했다. 3일 만에 대부분 시장 커버 학습을 완료하고 99% 정확도를 달성했으며, 최종 견적 전 사람이 검토하는 Human-in-the-loop 구조를 유지한다.

통신·인프라처럼 복잡한 B2B 가격 결정에 에이전틱 AI가 실제 운영 단계로 진입한 첫 대형 사례다.

## 📚 오늘의 주목 논문들

- **Memex(RL)**: 컨텍스트 창 한계를 넘기 위해 인덱스 기반 경험 메모리를 도입한 장기 LLM 에이전트 아키텍처
- **Inherited Goal Drift**: 강한 모델도 약한 에이전트의 궤적을 물려받으면 목표 이탈이 발생할 수 있다는 경고 — 멀티에이전트 설계의 안전장치 필요성을 강조
- **SWE-CI**: CI 파이프라인 내 에이전트 역량을 측정하는 새 벤치마크 — "한 번 고치기"에서 "지속 통합 환경 운영 안정성"으로 평가 기준이 높아진다
- **Cloudflare Markdown for Agents**: AI 크롤러가 HTML 대신 Markdown을 직접 요청 → 토큰 효율 최대 5배 향상
